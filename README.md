# QR Code Generator

A simple and responsive QR Code Generator built with HTML, CSS, and JavaScript. Enter text or a URL and instantly generate a QR code.

## Features

- Generate QR codes from text or URLs
- Press **Enter** or click the **Generate QR Code** button
- Input validation for empty fields
- Shake animation for invalid input
- Clean and responsive user interface

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- QR Code API

## How It Works

1. Enter text or a URL into the input field.
2. Click **Generate QR Code** or press **Enter**.
3. The application sends the input to a QR code generation API.
4. The generated QR code is displayed on the page.
5. If the input field is empty, a shake animation is triggered.
